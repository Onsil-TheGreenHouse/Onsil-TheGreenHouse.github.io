본격적으로 시작하기 전에

이 notebook안에서 matplotlib 패키지 하나만 설치해보겠습니다.

>!pip install matplotlib

notebook안에서 pip를 사용하려면 앞에 !(느낌표)를 붙이면 됩니다.

앞으로 실습을 하면서 설치가 안됬다! 라는 오류가 뜨면,

!pip를 이용해 설치해주시면 됩니다.

예를 들어 'no module named pandas' 라는 에러가 났다면

>!pip install pandas

를 입력해 pandas를 설치해주시면 됩니다.
