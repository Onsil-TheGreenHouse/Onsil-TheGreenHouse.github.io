## equals()와 hashCode() 작성하기

<br>

equals()와 hashCode()는 자바에서 객체가 같은지 체크하기 위해서

Object클래스에서 오버라이딩 해야하는 메소드입니다.

여러 IDE에서는 자동으로 이 두 메소드를 오버라이딩 해주는 기능이 있습니다.

자바의 대표적인 IDE인 Eclipse와 IntelliJ를 살펴보겠습니다.
